# my-fullpage

简单的Js fullpage插件

[Demo](http://showonne.github.io/my-fullpage/)

Need HTML structs
```
<div class='fullpage-cover'>
  <div class='wrapper'>
    <section>
      ...
    </section>
    <section>
      ...
    </section>
  </div>
</div>
```

In js file,initialize page.
```
FullPage(options)
```

options:
- animateDuration: default: 500
- animateType: default: 'ease'
- afterLoad: default: function(){}
